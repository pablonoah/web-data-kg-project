# Data Directory

## samples/
- `crawler_output.jsonl` — Raw crawled pages (cancer research domain)
- `extracted_knowledge.csv` — NER entities extracted by spaCy

## kge/
Generated by `src/kge/prepare_data.py`:
- `train.txt`, `valid.txt`, `test.txt` — Full dataset splits
- `entity2id.txt`, `relation2id.txt` — ID mappings
- `subset_20000/`, `subset_50000/` — Size sensitivity subsets

## Large Files
The expanded KB files (`expanded_kb.nt`, `final_kb.nt`) are excluded from git due to size.
To regenerate them, run:
```bash
python src/kg/expansion.py
python src/kg/merge_report.py
```
